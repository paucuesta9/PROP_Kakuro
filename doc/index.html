Directorio de Documentación

Este directorio contiene toda la documentación del proyecto. Se puede encontrar:

  • Directorio Doxygen (3a entrega): Documentación generada por doxygen
  • Documento Algoritmos y estructuras de datos (2a entrega): Fichero dónde se explican los algoritmos y estructuras de datos usados
  • Documento Casos de uso (1a entrega): Documento con el diagrama y explicación de los casos de uso
  • Documento Descripción juegos de prueba (3a emtrega): Docuemnto dónde se explican los casos de prueba de los drivers
  • Documento Diagrama de clases (2a emtrega): Documento que contiene los diagramas de clases de cada capa
  • Documento FirstDesign (3a entrega): Documento que contiene el diseño inicial de la capa de presentación
  • Documento librerias externas (3a entrega); Documento que contiene las librerias externas usadas
  • Documento listado funcionalidades (3a entrega): Documento que contiene las funcionalidades del proyecto indicando los extras
  • Documento manual de drivers y junit (3a entrega): Documento dónde se explican cómo usar los drivers y junit
  • Documento manual del usuario (1a entrega): Documento dónde se explica al usuario cómo usar la aplicación, compilar, ejecutar y las normas
  • Documento manual del usuario final (3a entrega): Documento dónde se explica al usuario cómo usar la aplicación en formato interfaz, compilar, ejecutar y las normas
  • Documento Relación de clases (3a entrega): Se indica que clases han sido desarrolladas por cada integrante del equipo
