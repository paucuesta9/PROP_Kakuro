Directorio de Documentación

Este directorio contiene toda la documentación del proyecto. Se puede encontrar:

  • Directorio Doxygen: Documentación generada por doxygen
  • Documento Algoritmos y estructuras de datos: Fichero dónde se explican los algoritmos y estructuras de datos usados
  • Documento Casos de uso: Documento con el diagrama y explicación de los casos de uso
  • Documento Descripción juegos de prueba: Docuemnto dónde se explican los casos de prueba de los drivers
  • Documento Diagrama de clases: Documento que contiene los diagramas de clases de cada capa
  • Documento manual de drivers: Documento dónde se explican cómo usar los drivers
  • Documento manual del usuario: Documento dónde se explica al usuario cómo usar la aplicación, compilar, ejecutar y las normas
  • Documento Relación de clases: Se indica que clases han sido desarrolladas por cada integrante del equipo
