Directorio Raíz

Este es el directorio raíz e incluye:

  • Directorio bin:Clases compiladas, preparadas para ejecutar
  • Directorio data:Se encuentran diferentes kakuros distribuidos por dificultad y tamaño. Además tambien se encuentran los inputs y outputs de los juegos de prueba
  • Directorio doc:Se encuentra toda la documentación del proyecto
  • Directorio lib: Se encuentran las librerias necesarias para JUnit
  • Directorio src: Se encuentran todas las clases, controladores, drivers y junits desarrollados
  • Doxyfile: Es el fichero para generar la documentación de Doxygen
  • Makefile: Es el fichero para compilar y ejecutar el programa, los drivers y Junit
  • MANIFEST.MD: Fichero para asignar el Main Class
  • README.MD: Es el fichero pedido con los nombres y correos de los integrantes del equipo
