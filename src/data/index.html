Data

Este paquete es el de la capa de persistencia e incluye las clases necesatias para poder acceder a archivos, guardar datos, leer datos, etc. y la comunicación con la capa de dominio

Los ficheros que se incluyen son los siguientes:

  • CtrlData.java

CtrlData.java

Esta clase es el controlador de la capa y que accede a los ficheros para leer, escribir, crear, eliminar, etc. y la comunicación con la capa de dominio

La clase es de tipo Singleton para que solo haya una sola instancia en toda la ejecución
