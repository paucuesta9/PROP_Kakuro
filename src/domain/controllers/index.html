Paquete controllers

En este paquete podemos encontrar los controladores de la capa de dominio. Además se incluyen tambien los directorios para los drivers, para los JUnits y para los stubs. En concreto, podemos encontrar:

  • Directorio drivers
  • Directorio junits
  • Directorio stubs
  • CtrlDomain.java
  • CtrlGenerate.java
  • CtrlPlay.java
  • CtrlPlayer.java
  • CtrlResolve.java
  • CtrlValidate.java

Directorio drivers

En este directorio podemos encontrar los diferentes drivers

Directorio junits

En este directorio podemos encontrar los tests de junit

Directorio stubs

En este directorio podemos encontrar los stubs

CtrlDomain.java

Este controlador se encarga de la comunicación entre las capas de presentación y la capa de persistencia

CtrlGenerate.java

Este controlador se encarga del caso de uso de Generar nuevos Kakuros

CtrlPlay.java

Este controlador se encarga del caso de uso de Jugar una partida

CtrlPlayer.java

Este controlador se encarga de todo que lo que tiene que ver con los jugadores

CtrlResolve.java

Este controlador se encarga del caso de uso de Resolver un kakuro

CtrlValidate.java

Este controlador se encarga del caso de uso de Validar un kakuro
