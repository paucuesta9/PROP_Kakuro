Paquete Exceptions

En este paquete podemos encontrar las clases de las diferentes excepciones. En concreto podemos encontrar:

  • CantGenerateException.java
  • NoTypeCellException.java
  • PlayerExists.java
  • WrongPasswordException.java

CantGenerateException.java

Esta clase es la que contiene la excepcion de cuando se intenta jugar un kakuro de mas de 20 de tamaño y no existe

NoTypeCellException.java

Esta clase es la que contiene la excepcion de cuando se lee un kakuro con caracteres invalidos

PlayerExists.java

Esta clase es la que contiene la excepcion de cuando se intenta registrar un usuario que ya existe

WrongPasswordException.java

Esta clase es la que contiene la excepcion de cuando al iniciar sesion introduce una contraseña incorrecta
